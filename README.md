# My Portfolio

This is my personal portfolio website built with [Next.js](https://nextjs.org) to showcase my projects, skills, and experience as a software developer.  

The site is fully responsive, optimized for performance, and includes modern web development features.

## Features

- Built with **Next.js 16** using the App Router
- Styled with **CSS Modules** for scoped styling
- Fully **responsive** and mobile-friendly
- Optimized fonts using [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts)
- Interactive components including buttons, social links, and project sections
- Easy to customize and extend for future updates

## Getting Started

To run the development server locally:

```bash
npm install       # Install dependencies
npm run dev       # Run the development server
# or
yarn install
yarn dev
# or
